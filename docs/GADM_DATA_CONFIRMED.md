### 州/省級數據（Level 1）

```json
{
  "GID_0": "TWN",
  "COUNTRY": "Taiwan",

  "NL_NAME_1": "福建",
  "VARNAME_1": "Fukien"

```

**結論**：✅ **數據包含完整名稱字段**
- `NAME_1`: 英文名稱
- `NL_NAME_1`: 本地/中文名稱
- `VARNAME_1`: 變體名稱

### 縣市級數據（Level 2）

## ✅ 最終結論

**GADM 資料確實包含名稱字段，不是資料本身的問題。**

如果無法顯示名稱，問題可能在：
1. 數據加載流程
2. 屬性讀取邏輯
3. 調用鏈路
