# 标记工作流程重新设计

## 新的工作流程（Google Maps 风格）

1. **搜索位置** → 自动缩放到位置
2. **点击地图** → 显示图标选择弹窗
3. **选择图标** → 在该位置添加标记

## 需要修改的内容

### 1. 移除侧边栏图标选择器
- 不再在侧边栏显示图标选择器
- 改为点击地图后弹出选择

### 2. 修改搜索功能
- 搜索后自动缩放到位置
- 不直接添加标记
- 提示用户点击地图添加标记

### 3. 添加地图点击模式
- 检测是否在"添加标记"模式
- 点击地图后显示图标选择弹窗
- 弹窗显示常用图标（类似 Google Maps）

### 4. 创建图标选择弹窗
- 显示常用图标选项
- 点击图标后在该位置添加标记
- 关闭弹窗

## 实现步骤

1. 隐藏侧边栏图标选择器
2. 修改搜索功能：只缩放，不添加标记
3. 添加"添加标记模式"开关
4. 创建地图点击处理：显示图标选择弹窗
5. 创建图标选择弹窗 UI
6. 实现图标选择后添加标记

## 图标样式参考 Google Maps

- 使用 Google Maps 的标准颜色
- 使用标准的图标样式（pin、circle 等）
