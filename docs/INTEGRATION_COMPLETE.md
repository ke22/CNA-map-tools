## 🚀 **測試步驟**

### **1. 啟動本地服務器**

```bash
# 方法 1: Python
python3 -m http.server 8000

# 方法 2: 使用腳本
bash start-server.sh

# 方法 3: Node.js
node server.js
```

### **2. 打開瀏覽器**

訪問：
```
http://localhost:8000/index-enhanced.html
```

### **3. 測試功能**

1. **測試國家選擇**
   - 點擊「國家」按鈕
   - 在地圖上點擊國家（如台灣）
   - 應該可以選擇並上色

2. **測試第一級行政區**
   - 點擊「第一級行政區」按鈕
   - 等待載入（第一次載入可能需要幾秒）
   - 在地圖上點擊區域（如台北市）
   - 應該可以選擇並上色

3. **測試第二級行政區**
   - 點擊「第二級行政區」按鈕
   - 等待載入（文件較大，可能需要更長時間）
   - 在地圖上點擊區域（如信義區）
   - 應該可以選擇並上色

## 🐛 **如果遇到問題**

### **問題：文件載入失敗**

**解決方案：**
1. 檢查本地服務器是否運行
2. 確認文件路徑正確：`data/gadm/optimized/`
3. 檢查瀏覽器控制台錯誤

### **問題：404 錯誤**

**解決方案：**
- 確保服務器從項目根目錄啟動
- 檢查文件是否在正確位置

### **問題：CORS 錯誤**

**解決方案：**
- 使用本地服務器（`python3 -m http.server`）
- 不要直接用 `file://` 打開 HTML 文件
