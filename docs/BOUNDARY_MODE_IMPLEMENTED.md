# 邊界模式實現完成

## ✅ **已完成**

### **UI 更改：**
- 只有兩個按鈕：
  - **國家** - 只顯示國家邊界
  - **行政區** - 同時顯示第一級和第二級行政區

### **邏輯更改：**

**國家模式：**
- 只加載和顯示國家邊界（Level 0）
- 點擊時只能選擇國家

**行政區模式：**
- 同時加載和顯示第一級和第二級行政區（Level 1 + Level 2）
- 點擊時自動檢測是第一級還是第二級
- 可以分別選擇並上色

### **技術實現：**

1. **狀態管理：**
   - `currentAreaType: 'country' | 'administration'`
   - `administrationLevel: 'state' | 'city' | null`（點擊時自動檢測）

2. **切換邏輯：**
   - 選擇「國家」：只顯示 Level 0
   - 選擇「行政區」：同時顯示 Level 1 和 Level 2

3. **點擊檢測：**
   - 在行政區模式下，優先檢測第二級（城市），然後第一級（州）
   - 自動設置 `administrationLevel`

---

## 🎯 **使用方式**

1. **選擇國家模式：**
   - 點擊「國家」按鈕
   - 只能選擇國家邊界

2. **選擇行政區模式：**
   - 點擊「行政區」按鈕
   - 可以選擇第一級或第二級行政區
   - 點擊時自動識別級別
