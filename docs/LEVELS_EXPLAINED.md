# 邊界層級說明

## 📋 **您需要的層級**

根據您的需求：
- **Level 0:** 國家（國家邊界）
- **Level 1:** 第一級行政區（省/州/直轄市）
- **Level 2:** 第二級行政區（縣/市/區）- 可選

這樣您就有三個層級可以使用：
1. 只顯示國家邊界
2. 顯示國家 + 第一級行政區
3. 顯示國家 + 第一級行政區 + 第二級行政區

---

## 🎯 **實際應用**

例如台灣：
- **Level 0:** 台灣（國家）
- **Level 1:** 台北市、新北市、桃園市...（第一級行政區）
- **Level 2:** 信義區、中正區...（第二級行政區）

這樣您就可以：
- 選擇台灣並上色
- 選擇台北市並上色
- 選擇信義區並上色
