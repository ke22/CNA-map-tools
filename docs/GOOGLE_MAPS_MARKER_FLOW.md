# Google Maps 风格的标记工作流程

## 新的工作流程

1. **搜索位置** → 自动缩放到位置（不直接添加标记）
2. **点击地图** → 在地图点击位置显示图标选择弹窗
3. **选择图标** → 在该位置添加标记

## 需要实现的功能

### 1. 修改搜索功能
- 搜索后只缩放到位置，不添加标记
- 提示用户"点击地图添加标记"

### 2. 添加地图点击检测
- 检测是否在"标记模式"
- 点击地图后获取坐标
- 显示图标选择弹窗

### 3. 创建图标选择弹窗
- 类似 Google Maps 的图标选择器
- 显示常用图标（红色、蓝色、绿色等）
- 点击图标后添加标记

### 4. 隐藏/移除侧边栏图标选择器
- 不再在侧边栏显示图标选择器
- 改为点击后弹窗选择

### 5. 图标样式参考 Google Maps
- 使用 Google Maps 的标准颜色
- 使用标准的图钉样式
