# 層級更新說明

## 🎯 **您的需求**

您說：「層級不對，應是 國家 跟 行政區（可分兩級） 就好」

所以我理解為：
- **國家** (Country) - Level 0
- **行政區可分兩級：**
  - **第一級行政區** (Level 1) - 如：省、州、直轄市
  - **第二級行政區** (Level 2) - 如：縣、市、區
