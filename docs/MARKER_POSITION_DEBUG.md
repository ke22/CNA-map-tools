# 标记位置问题调试

## 当前实现状态

代码中同时使用了两种方法：
1. 直接修改元素尺寸（width/height）- 可能导致锚点偏移
2. CSS transform scale - 应该更准确

## 问题描述

从图片看，标记在缩放后位置可能不准确。

## 需要的信息

请告诉我：
1. 标记是否在缩放时移动了位置？
2. 标记的尖点（pin tip）是否偏离了实际坐标？
3. 缩放时标记大小是否正确变化？

## 可能的解决方案

1. **使用 transform scale 而不是修改尺寸**
   - 保持元素实际尺寸不变
   - 只使用 CSS transform scale
   - 这样锚点计算不会受影响

2. **改进锚点设置**
   - 对于旋转的 pin，可能需要调整 anchor
   - 或使用 offset 来微调位置

3. **禁用缩放功能**
   - 如果缩放导致问题，可以先禁用
   - 保持标记大小恒定

请提供更多具体信息以便精确修复！
