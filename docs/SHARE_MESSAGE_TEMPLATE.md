# 分享信息模板

## 📤 可以直接复制使用：

---

**给开发者的分享信息：**

```
# CNA Map Tools - 地图工具

一个功能完整的地图工具，支持区域标注、标记添加和 AI 辅助分析。

🚀 快速开始（3步）：

1. 克隆仓库：
   git clone https://github.com/ke22/CNA-map-tools.git
   cd CNA-map-tools
   git checkout feature/genai

2. 配置 Mapbox Token：
   - 访问 https://account.mapbox.com/access-tokens/
   - 注册/登录账号（免费）
   - 复制 Default public token
   - 打开 config.js，替换 MAPBOX.TOKEN

3. 启动应用：
   ./SETUP_API_KEY.sh    # 可选：设置 AI 功能
   ./START_SERVER.sh      # 启动服务器
   或直接打开 index-enhanced.html

✨ 主要功能：
- 🗺️ 多地图样式切换
- 🎨 国家/行政区选择和上色
- 📍 标记添加和管理
- 🤖 AI 辅助分析（可选）
- 📥 地图导出（PNG/JPG）

📚 详细文档：
- GETTING_STARTED.md - 快速开始
- SETUP_GUIDE.md - 详细设置指南
- README_FOR_SHARING.md - 完整功能说明

仓库：https://github.com/ke22/CNA-map-tools
分支：feature/genai
```
