## 常见问题

### 1. 显示 "Unknown Country"
- **原因**: 国家名称获取失败
- **检查**: Console 中查看 `getAreaName` 的日志
- **解决**: 检查 `COUNTRY` 属性或 `GADM_LOADER`

### 2. 点击后没有反应
- **原因**: 边界未加载或点击位置不准确
- **检查**: 确保边界图层已加载
- **解决**: 放大后再点击，或直接点击边界线

### 3. 颜色没有应用
- **原因**: 图层创建失败
- **检查**: Console 中查看错误信息
- **解决**: 检查图层创建逻辑
