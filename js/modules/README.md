# Feature Modules

功能模块，每个模块负责一个独立的功能领域。

## boundary/
边界相关功能：
- BoundaryLoader.js: 边界数据加载
- BoundaryRenderer.js: 边界渲染
- BoundarySelector.js: 边界选择

## labels/
标签相关功能：
- LabelManager.js: 标签管理
- LabelRenderer.js: 标签渲染

## markers/
标记相关功能：
- MarkerManager.js: 标记管理
- MarkerRenderer.js: 标记渲染

## ai/
AI 相关功能：
- AIService.js: AI 服务封装
- AIResultProcessor.js: AI 结果处理

## data/
数据相关功能：
- GADMLoader.js: GADM 数据加载
- MapboxLoader.js: Mapbox 数据加载
- DataValidator.js: 数据验证
